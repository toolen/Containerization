Образы контейнеров

Dockerfile

            FROM python:3.7-alpine
            WORKDIR /app
            COPY . /app
            RUN pip install -r requirements.txt
            EXPOSE 80
            ENV NAME World
            CMD ["python", "app.py"]

Docker контейнер

Сеть

  • Bridge
  • Host
  • None

Хранение и управление данными

  • В контейнере
  • Привязка монтирования (mount)
  • Тома (volumes)
  • tmpfs

Тома (volumes)

  • Резервирование и преносимость
  • Управление через CLI или Docker API
  • Работа с Linux и Windows контейнерами
  • Распределение между несколькими контейнерами
  • Хранение на удаленных узлах или в облаке, шифрование и другие функции
  • Предварительное заполнение контейнером

Угрозы безопасности

  • Уязвимости в ядре хоста
  • DDoS через монополизацию ресурсов
  • Взлом контейнера, повышение привилегий
  • Использование зараженных образов
  • Утечка секретов

Docker compose

docker-compose.yml

            version: '3'
            services:
                web:
                    build: .
                    ports:    
                        - "5000:5000"
                redis:
                    image: "redis:alpine"

Преимущества docker compose

  • Множество изолированных сред на одном хосте
  • Сохранение томов при перезапуске контейнеров
  • Пересоздание только измененных контейнеров
  • Поддержка переменных сред
